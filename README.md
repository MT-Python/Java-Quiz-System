Java  Quiz System


##  Features

### Admin
- Login as admin
- Add Multiple Choice Questions (MCQs)
- Questions stored in `quiz.json`

###  Student
- Login as student
- Attempt 10 random quiz questions
- 1 mark per question
- No negative marking
- Final score and result message shown

### Technologies Used
- Java (Oracle OpenJDK 21)
- Gradle
- Gson (JSON handling)
- IntelliJ IDEA
